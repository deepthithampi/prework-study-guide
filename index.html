HTML

  • HTML (Hypertext Markup Language) is used to create the structure of a webpage.
  • Elements are used in designing a webpage. They are the building blocks of a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes provide additional info about an element.They are placed within the opening tag of an element.

Git

  • Git is powerful version control system.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • pull requests are used to propose changes to the main codebase.
  • pushing means updating our changes to a remote repo
  • pulling means fetching changes from the remote repo to our local repo
